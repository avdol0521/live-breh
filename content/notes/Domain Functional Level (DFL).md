- the [[Domain Functional Level (DFL)]] controls what AD features are available based on the oldest windows server version present in the domain 
	- for example if a domain has windows server 2008 and 2016 domain controllers, the DFL cant go higher than windows server 2008
- another thing to know about is the [[Forest Functional Level (FFL)]]
