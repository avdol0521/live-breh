### common commands: 
- `cd` shortcuts:
	- can do just / for root
	- can also do username to jump to home
- `ls la`
	- lists all files with [[kali file perms and stuff]] listed out
- `>` to insert and `>>` to append
- `passwd`
	- changes pass for a user 
	- `sudo passwd root` 
- `xfwm4`
	- to troubleshoot UI tweaking
- `rm` variants:
	- `rm -r dirname`
